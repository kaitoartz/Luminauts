## Brand & Style
Tema = Espacio suave. LumiNauts reemplaza EduPlay. Usuarios = Ludonáuticos. Diseño Soft Cosmic. Premium, intencional, cero neón. Fondos oscuros suaves (carbón/azul noche), formas ultra redondeadas.

## Colors
Adiós primarios duros.
- **Primary:** Azul Polvo Espacial (#6B8BB4).
- **Secondary:** Celeste Suave (#8DA9C4).
- **Tertiary:** Lavanda Muted (#E0B0FF) para logros.
- **Background:** Cielo Nocturno (#141923). Sin negro puro.
- **Cuentas/Logros:** Estrellas crema, feedback verde suave.

## Typography
Nunito o Quicksand. Letras redondas, amigables, seguras para niños, legibles para Comandantes (profesores). Sin pesos agresivos.

## Layout & Spacing
Grid 12-col desktop, 4-col mobile. 8px base.
- **Bitácora (Dashboard):** Grid de Planetas.
- **Mapa Estelar (Game View):** Botón "Volver a Nave" accesible.
- **Gutters:** 24px para respirar en el espacio.

## Elevation & Depth
Sombras Nebula. Elementos flotan en baja gravedad.
- **Level 0:** Espacio profundo (#141923).
- **Level 1:** Órbita. Sombra muy difuminada color primary.
- **Level 2:** Hover. Brillo sombra suave.
- **Level 3:** Modals. Frosted glass oscuro.

## Shapes
Cero bordes afilados.
- **Cards:** 24px radius.
- **Botones:** Pill-shape (full) o 16px.
- **Chips:** Pill-shape.

## Components
### Planet Cards
Ilustración planeta 16:9 bordes suaves. Botón "Lanzamiento".
### Constellation Counter
HUD flotante. Píldora con estrella crema. Pulso suave al ganar estrella.
### Buttons
Primary azul suave. Call-to-action lavanda.
### Input Fields
Fondo oscuro suave, borde tenue. Hover/Focus = glow nebula suave.
